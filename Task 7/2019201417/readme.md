# Task 7

任务完成情况如下：
* 使用指针与动态内存分配。
* 使用 `make` 管理多文件编译。
* （选做）将二叉搜索树的实现编译为共享库（`.so` 文件），并在程序中对其进行调用。

使用以下命令：
```bash
# cd "Task 7/2019201417王子恒/"
$ make
$ ./main
（开始你的输入）
```

在上述命令的基础上，使用以下命令：
```bash
$ ./debug_main
（开始你的输入）
```
