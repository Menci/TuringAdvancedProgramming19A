# Task 8

哈夫曼编码压缩。

使用以下命令：
```bash
# cd "Task 8/2019201417王子恒/"
$ make K=? run??
```
运行 ??.cc 生成数据，对其压缩并检验，新的压缩文件为my_zip.bin。

其中 `?` 为 ??.cc 中参数 `k` 的数值。

其中 `??` 为 .cc 文件的编号。

使用以下命令：
```bash
# cd "Task 8/2019201417王子恒/"
$ make K=? special
```

压缩数据zipped.bin并检验，新的压缩文件为my_zip.bin。

其中 `?` 为你zipped.bin中参数 `k` 的数值。

最后可通过以下命令将生成文件删除：
```bash
# cd "Task 8/2019201417王子恒/"
$ make clean
```
